General

This is where the program starts. In the upper left hand side is a blue shaded area. This is the banner, where all work is shown.

The File menu is just the common commands for save, save as, new, open and exit. The Help menu contains this help file menu and the About dialog.

To start select the size of banner you want from the "Banner Sizes" list box.. If you can't find a standard one to suit you at the top you will find "- Custom -". Selecting this brings up a custom dialog box where you can make it the size you want.

Next if you want a solid color for a background press the "Solid" option button and press "Select Color" button for the color dialog.

If you want a graphical background press the "Graphic" option button and then press "Browse" to find the graphic you want.
