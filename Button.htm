Button

This is just a collection of buttons. I added this just to show how buttons can be changed. It is only applicable to programmers.

Standard Button

The normal button.

Graphic Button

This is the standard button with the style set to 1-Graphical. As you can see the caption drops to the bottom of the button. Which in most cases you do not want.

You can add a graphic by selecting the picture property and loading the graphic.

Graphic Button and No Caption

Again the standard button with style set to 1-Graphical and no caption. Load your picture with the picture property. The button does not have a stretch property so you have to either size the button to your graphic or size the graphic to your button.

Image Control

Because the image control has a stretch property you can resize the image automatically and set the size of your button to whatever you want. The button appearance is handled with pictures. One picture has the light border on the left and top sides is the button up and the light side on the right and bottom is the button down. On the image button down procedure load the button down graphic. On the image button up procedure load the button up graphic. The button up graphic is loaded on start up.
